Guestbook

  • A 2011 MacBook Air is used as the server for this guestbook.
  • It runs Debian Linux with XFCE as the desktop environment.
  • Since the internal SSD no longer works, a 32GB SD card is used as boot and sole storage.
  • The machine is always plugged in because the battery holds no charge.
  • The full-stop and comma keys were damaged by a tea spill. These are remapped using Xmodmap to alternative keys (caps lock and plus/minus).
  • The web stack consists of a PHP-based API, Nginx handling HTTP requests, and SQLite for storing guestbook comments.
  • A Cloudflare tunnel is used to securely expose the API to the internet.
  • Vim is my text editor of choice.
  • Be nice in the comments and please don't hack me.
