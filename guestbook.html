Ianto Cannon's guestbook

This guestbook runs on a 2011 macbook air using Debian Linux with the XFCE desktop environment. The internal SSD is no longer functional, so a 32GB SD card is mounted as the storage and boot drive. The laptop is always plugged in because the battery is toast. The full-stop and comma keys were damaged by a tea spill and are remapped using Xmodmap to the caps lock and plus/minus keys, which I never used anyway. We have Nginx serving an API written in PHP, and the comments are stored in an SQLite database.

Be nice in the comments and please don't hack me.
